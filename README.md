# image_searcher
누구나 쉽게 나만의 이미지 검색 엔진 만들수 있습니다.

○ 개발 목표

- 누구나 손쉽게 이미지 검색을 통한 상품 추천 서비스를 사용가능하도록 API 제공

○ 주요 개발 사항

- CNN을 이용한 이미지 분류 학습 모듈

- Django Framework를 통한 RESTful API 서버

○ 학습 데이터

-  사용자가 원하는 만큼 zip파일 형태로 업로드
 
○ 학습량

- 100 epoch, 평균 1분
